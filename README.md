# 🏠 DEALAK - تطبيق عقاري متكامل

منصة متكاملة لبيع وإيجار العقارات في سوريا

## 📋 نظرة عامة

DEALAK هو تطبيق عقاري شامل يهدف إلى تسهيل عملية بيع وإيجار العقارات في سوريا من خلال منصة رقمية متكاملة تجمع بين تطبيق الهاتف الذكي والموقع الإلكتروني.

## 🎯 الميزات الرئيسية

- 🔍 **البحث المتقدم:** البحث عن العقارات حسب الموقع، المساحة، السعر، والنوع
- 📍 **تحديد الموقع:** استخدام GPS لتحديد موقع العقار بدقة
- 💬 **المحادثة الفورية:** التواصل المباشر بين البائع والزبائن
- 🗺️ **خرائط Google:** عرض العقارات على الخرائط التفاعلية
- 🔔 **نظام الإشعارات:** إشعارات فورية للطلبات والعقارات الجديدة
- 💳 **الدفع الإلكتروني:** دفع آمن ومحمي
- 👤 **إدارة الحسابات:** تسجيل دخول آمن وإدارة ملفات المستخدمين

## 👥 المستخدمون

### الزبائن (Customers)
- البحث عن العقارات
- إرسال طلبات الشراء/الإيجار
- التواصل مع البائعين
- تتبع الطلبات والصفقات

### البائعون (Sellers)
- نشر العقارات للبيع/للإيجار
- إضافة صور وتفاصيل
- استلام وإدارة الطلبات
- التواصل مع الزبائن

### مدير النظام (Admin)
- إدارة المستخدمين
- مراقبة الإعلانات
- تتبع الصفقات والدفعات
- إدارة الإعدادات والخصومات

## 🗄️ قاعدة البيانات

### الكيانات الرئيسية

- **Users:** المستخدمون (بائعون وزبائن)
- **Property:** العقارات
- **Picture:** صور العقارات
- **Notice:** الإشعارات
- **Deal:** الصفقات
- **Payment:** الدفعات
- **Discount:** الخصومات
- **Request:** الطلبات
- **Deal Information:** معلومات الصفقة
